
Для определения памяти, которую занимает любой объект java, нужно понимать из чего состоит любой объект, а также понимать, сколько разрядов у системы.

Для начала определим из чего состит объект.

Объект:
 Если речь идет о 32-х разрядной системе, то размер заголовка — 8 байт, если речь о 64-х разрядной системе, то соответственно — 16 байт.
 
 - Заголовок Объекта
   * HashCode
   * Garbage Collection Information
   * Type Information Block Pointer
   * Lock
   * Array Length 
 
 - Память для приметивных типов
 
 - Память для ссылочных типов
 
 - смещение/выравнивание для того чтобы размер объекта был кратен 8 байтам
 
 Соотвесвенно, если объект у нас пустой, память, которую он занимается расходуется в основном только на его заголовок.
